# 💻 Microsoft SDE Full-time Interview Experience

Here is the breakdown of my interview experience for the Software Engineer role at Microsoft during the campus placements.

## 📝 Round 1: Coding Assessment (Online)
The test was hosted on Codility. There were 3 questions to be solved in 90 minutes.
*   **Q1:** String manipulation and greedy approach (Easy-Medium).
*   **Q2:** Array segment search using hash maps (Medium).
*   **Q3:** Dynamic Programming on tree heights (Hard).
I managed to solve all three questions with all test cases passing.

## 👥 Round 2: Technical Interview (Virtual)
This focused heavily on data structures and algorithms. The interviewer asked me to share my screen and code in a shared editor.
*   **Problem:** Find the lowest common ancestor in a Binary Tree (not BST).
*   **Follow-up:** Optimize the solution to use space complexity of O(1) without parent pointers.
*   *Verdict:* The interviewer seemed satisfied with the optimization.

## 🛠️ Round 3: System Design & Core CS Concepts
Deep dive into my internship projects followed by a system design question.
*   **Design Topic:** Design a URL Shortener service (like Bit.ly).
*   **Key Discussions:** Database partitioning, caching strategies (Redis), and dealing with concurrency.
*   **CS Fundamentals:** Questions on DBMS transactions (ACID properties) and Operating Systems page replacement policies.

## 🤝 Round 4: AA Round (Asappropriate/HR)
Standard behavioral round focusing on teamwork, handling conflict, and alignment with Microsoft's growth mindset culture.

*Verdict: Selected!*
